Тестовое задание в компанию T1 Консалтинг

На вход подаётся строка через параметр get запроса "string".
Возвращаются символы по из строки и количество их количество.
Пример запроса: http://localhost:8080/?string=aabbcccc
Возвращаемое значение отсортировано в порядке убывания количества символов,
при их равенстве по символам в алфавитном порядке.

Java 8+
Spring boot 2+
